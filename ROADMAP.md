# 5-Day Hackathon Roadmap

## Day 1: Foundation

- Create repo structure.
- Write core documentation.
- Define MVP agent contracts.
- Prepare synthetic data and knowledge base placeholders.

## Day 2: Backend Pipeline

- Build local FastAPI backend.
- Create a basic war room pipeline.
- Add placeholder agent execution interfaces.
- Return structured responses for frontend integration.

## Day 3: Reasoning Engines

- Add readiness scoring.
- Add failure prediction.
- Add debate and critic flow.
- Add what-if simulation stubs.

## Day 4: Frontend and Foundry IQ

- Build frontend dashboard screens.
- Connect dashboard to backend endpoints.
- Add Foundry IQ mock or integration.
- Show citations and grounded evidence.

## Day 5: Demo and Polish

- Test end-to-end demo path.
- Prepare deployment files.
- Finalize demo script and pitch.
- Rehearse judging narrative.
