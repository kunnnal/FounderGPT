# Agents

## MVP Agents

### Planner Agent

Breaks the founder's startup idea into analysis tasks, selects which specialist agents should run, and defines the expected output structure.

### Market Analyst Agent

Evaluates target customers, market timing, competitive landscape, positioning, and initial demand signals.

### Finance Agent

Reviews pricing, cost assumptions, runway needs, revenue potential, and basic unit economics.

### Risk Agent

Identifies major execution, market, product, financial, legal, and operational risks that could cause the startup to fail.

### Decision Agent

Synthesizes agent findings into a final readiness score, key risks, next actions, and a recommendation.
