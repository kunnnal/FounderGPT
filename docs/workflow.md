# Workflow

FounderGPT follows a structured reasoning flow:

User idea -> Planner -> Specialist Agents -> Debate/Critic -> Readiness Score -> Failure Prediction -> What-if Simulation -> Decision Report

## Flow Details

1. The founder submits a startup idea and context.
2. The Planner Agent decomposes the idea into analysis tasks.
3. Specialist agents evaluate the idea from market, finance, product, GTM, risk, and compliance angles.
4. Debate and critic steps challenge assumptions and expose conflicting viewpoints.
5. The readiness engine calculates an MVP-stage startup readiness score.
6. The failure prediction engine identifies likely failure modes.
7. The simulation engine runs what-if scenarios against key assumptions.
8. The Decision Agent and Report Agent produce the final founder-facing roadmap.
