# Project Goal

FounderGPT is a multi-agent startup war room for founders who need structured reasoning, not generic chat. It is designed to help evaluate startup ideas, expose risks, debate tradeoffs, simulate scenarios, and generate a practical execution roadmap.

FounderGPT is not a simple chatbot. The intended system coordinates multiple specialized agents that reason from different perspectives, challenge each other, retrieve grounded knowledge, and converge on a decision-ready report.

## Core Capabilities

- Multi-agent reasoning across market, product, finance, risk, and decision strategy.
- Agent debate to surface disagreement before final recommendations are made.
- Readiness scoring to estimate whether an idea is prepared for validation, MVP, launch, or fundraising.
- Failure prediction to identify likely causes of startup breakdown.
- What-if simulation for pricing, market, runway, GTM, and product assumptions.
- Grounded retrieval through Microsoft Foundry IQ and a curated knowledge base.

## Hackathon Objective

Deliver a focused MVP that demonstrates a believable multi-agent reasoning loop, clear Microsoft platform alignment, and a useful founder-facing dashboard.
