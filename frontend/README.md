# Frontend

Next.js frontend for the FounderGPT demo.

## What It Includes

- `/` landing page
- `/war-room` founder intake and live multi-agent output
- `/report/[sessionId]` saved report page backed by the FastAPI session endpoint

## Run

```powershell
cd frontend
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build Check

```powershell
cd frontend
npm run build
```
